MTI RFID ME GUI APP is an Android app to help people to evaluate MTI UHF RFID reader, including RFID ME USB Dongle and MINI ME readers.

Here includes the document as below.

1. The Reference Manual of MTI Low Level Command Set to control MTI UHF RFID readers.
	- MTI RU-888 RFID Module Command Reference Manual v5.0.pdf
	- MTI RU-827 RFID Module Command Reference Manual v1.2.pdf

2. The .apk file of MTI RFID ME GUI APP
	- MTI RFID ME GUI_v1.4.1.apk
	- MTI RFID ME GUI_v1.5.0.apk

3. The source code of MTI RFID ME GUI (for Android 3.7 or 4.0.1 above)
	- under folder \MTI RFID ME GUI_v1.4.1

4. RFID ME License and Warranty Agreement to use the provided document.
	- RFID ME License and Warranty Agreement Rev B 120629.pdf

Upon the acceptance on the RFID ME License and Warranty Agreement, it is welcome to build your own application by using the information we provided to work on the MTI UHF RFID module and reader.
